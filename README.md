# Interstellar V6

This is a BETA version of Interstellar rewritten from the ground up.

## Development

```sh
pnpm i
pnpm disable # Optional: Disables Astro's Telemetry - https://astro.build/telemetry
pnpm dev
```

## Production

```sh
pnpm i
pnpm disable # Optional: Disables Astro's Telemetry - https://astro.build/telemetry
pnpm build
pnpm start
```
